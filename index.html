Lexington Girls Who Code

We are an only girls coding group in Lexington, Massachusetts. Our aim is to learn as much as we can and to try

and get as many girls interested in coding as possible.

We learn new coding languages and we also learn how to work in groups and how to find a compromise when disagreeing.

Here are the languages we know and use:

JavaScript

Python

HTML (Hypertext Markup Language)

CSS (Cascading Style Sheets)
